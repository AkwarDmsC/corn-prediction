## v4.14 优化记录（2026-05-21）

> 2026-05-21校准报告：日盘方向准确率22%（基于9个样本，样本极少仅供参考）
> v4.14修复（均线空头+季节性禁用+区间调整）后待更多样本验证

### 问题根因
- **方向系统性错误**：季节性权重在5月总是正值(+0.1×5×0.3=+0.15)，当其他指标偏空时仍把整体方向向上拽，导致所有预测输出"偏强"
- **区间命中率偏低**：±6σ覆盖约82%，但实际只有75%（近期波动结构变化）
- **高价/低价预测偏保守**：ML模型低估日内波幅

### 修复内容

| 修复项 | 原值 | 新值 | 效果 |
|--------|------|------|------|
| `season_w` | `0.3`（固定） | `0.3 if ma_dir >= 0 else 0.0` | 均线空头时禁用季节性向上偏置 |
| `scen_d2` | `max(vol×6, 15)` | `max(vol×7, 18)` | 区间覆盖~89%（原75%→目标89%） |
| `_make_pred`方向 | `↗ if direction>0 else ↘` | 均线空头时强制↘ | 趋势确认机制，避免反向输出 |

### 日盘预测方向修复效果（2026-05-21实测）
- 修复前：↗ 震荡偏强（被季节性+0.15硬拽）
- 修复后：↘ 震荡偏弱(均线空头)（均线空头排列，趋势确认覆盖季节性）

## 四、技术面指标

### 均线系统

| 指标 | 周期 | 用途 |
|------|------|------|
| MA5 | 5日 | 短期趋势 |
| MA10 | 10日 | 辅助判断 |
| MA20 | 20日 | 中期趋势 |
| MA60 | 60日 | 长期趋势 |

**均线状态判断：**
- 多头排列：MA5 > MA10 > MA20 → 上升趋势
- 空头排列：MA5 < MA10 < MA20 → 下降趋势
- 混乱排列：趋势不明

### 布林带（20日）

```
上轨 = MA20 + 2×标准差
中轨 = MA20
下轨 = MA20 - 2×标准差
```

| 价格位置 | 信号 |
|---------|------|
| > 80%（接近上轨） | 超买区，警惕回调 |
| < 20%（接近下轨） | 超卖区，留意反弹 |
| 中轨上方 | 多头控盘 |
| 中轨下方 | 空头控盘 |

**关键价位预警（v4.3）：**
- 近5日高点（阻力）：突破视为多头信号
- 近5日低点（支撑）：跌破视为空头信号
- 价格 > 布林上轨 + 成交量放大 → 加速上涨信号
- 价格 < 布林下轨 + 缩量 → 空头陷阱警惕

|------|------|

## 五、基本面因素

### USD/CNY 汇率

| 方向 | 影响 |
|------|------|
| 人民币升值（USD/CNY ⬇️） | 利多玉米（进口成本下降） |
| 人民币贬值（USD/CNY ⬆️） | 利空玉米（进口成本上升） |

### INE原油（SC0）

| 方向 | 影响 |
|------|------|
| 原油上涨 ⬆️ | 利多玉米（乙醇生产需求） |
| 原油下跌 ⬇️ | 利空玉米 |

### CFTC美国玉米净持仓

- 每周五发布，反映国际机构对CBOT玉米的多空态度
- 净多增加 → 国际看多情绪升温（间接传导至DCE）
- 净空增加 → 国际看空情绪升温

### CBOT玉米（外盘）

- CBOT是全球玉米定价中心，影响中国进口成本
- CBOT当日涨跌方向传导至DCE概率约54%
- CBOT隔夜涨跌是DCE日盘开盘的重要参考

### BCI波罗的海运费指数（v4.8新增）

- 数据来源：`akshare.macro_china_freight_index`（BCI，好望角型船运价指数）
- BCI > MA5 → 运费上涨 → 进口成本上升 → 利多国内玉米
- BCI < MA5 → 运费下跌 → 进口成本下降 → 利空国内玉米
- **注意：BCI与DCE玉米方向一致率约48%，信号偏弱，权重设为0.3x**
- 当前：BCI=5139，MA5=4738（2026-05-07），上涨趋势

### DCE持仓量变化（v4.8新增）

- 数据来源：`futures_zh_daily_sina(symbol="C0")` 返回的 `hold` 字段
- 持仓量增加 → 资金入场，趋势延续概率大
- 持仓量减少 → 资金离场，趋势衰竭信号
- 变化 > ±1% 视为明显信号（权重0.5x）

> 接入方式：`fetch_enso()` 函数，直接抓取 NOAA 公开数据接口

### 预测置信度

| 等级 | 置信度 | 含义 |
|------|--------|------|
| 🟢 高 | > 70% | 指标高度一致，趋势信号清晰 |
| 🟡 中 | 40-70% | 信号混合，需关注关键价位突破 |
| 🔴 低 | < 40% | 指标严重分歧，趋势不明，谨慎操作 |

**信号一致性计算：**
```
一致性分数 = Σ(信号方向×权重) / Σ|权重|  → 归一化到 -1 到 +1
置信度 = |一致性分数| × 100%
```

## 十一、多周期预测

### 第二日预测（下一交易日）

```
成交量权重：
  放量(>1.2x) → 1.0（动能可靠）
  正常(0.8~1.2x) → 0.8
  缩量(<0.8x) → 0.5（动能削弱）

MACD动能 = MACD柱状图 × 2
RSI修正：
  RSI>65: -(RSI-65) × 1
  RSI<35: (35-RSI) × 1
  其他: (50-RSI) × 0.1

大豆权重 = 0.5（相关性>0.5时提升至0.5）
季节性权重 = 0.3

基准预测 = 最新价 + (MACD动能×成交量权重 + RSI修正) × 大豆权重 + 季节性分数×5×季节性权重
乐观/悲观 = 基准 ± 近期波动率×6

布林带硬约束：预测价不超过布林上下轨
```

### 短期（1-4周）

参考：MA20趋势、RSI区域、布林带位置、成交量趋势、政策事件

### 中期（1-3个月）

参考：MA60方向、豆粕相关性趋势、政策事件窗口、季度季节性

### 长期（3-12个月）

参考：年均价中枢趋势、多年政策框架、结构性供需

## 🌙 夜盘ML预测模型 v1.0（2026-05-19）

### 背景与数据

- **目标变量**：`night_change = T+1_open - T_close`（作为真实夜盘收盘的代理）
  - 原因：akshare 5分钟K线仅支持回溯至2026-04-23，真实夜盘样本仅14条，不足训练
  - 代理目标基于21年日线数据（共5172个样本）
- **交叉验证**：用14条真实夜盘数据评估模型

### 模型架构

| 模型 | 用途 | CV MAE | 方向准确率 |
|------|------|---------|-----------|
| Ridge回归 | 方向信号（系数可解释） | 6.24元/吨 | **69%** |
| RandomForest | 非线性特征捕捉 | 6.88元/吨 | 54% |
| **Ensemble** | Ridge 40% + RF 60% | — | 54% |
| **校准后** | Ridge方向 + 线性幅度校准 | 5.12元/吨 | 62% |

### 幅度校准

基于13条真实夜盘数据拟合：
