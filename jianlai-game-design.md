# 《剑来》书评文字修仙游戏 — 设计文档

> 基于问道长生 PWA 框架改造，纯 HTML/CSS/JS，竖屏

---

## 一、世界观设定

- 游戏世界：浩然天下（主线），随境界推进解锁其他天下
- 时间线：从陈平安少年时期开始，以原著章节为剧情节点
- 核心驱动：修炼 → 变强 → 探索更多原著剧情 → 突破境界

---

## 二、境界体系（完整复刻原著）

### 修炼体系（凡人→武道→练气）

| 阶段 | 大境界 | 小境界数 | 解锁内容 |
|------|--------|---------|---------|
| 凡人 | — | — | 初始，可锻体 |
| 武道 | 铜皮境 | 3层 | 解锁基础剑术 |
| 武道 | 锻体境 | 3层 | 解锁炼体功法 |
| 武道 | 归真境 | 3层 | 解锁内功心法 |
| 练气 | 练气境 | 9层 | 解锁飞剑、符箓 |
| 练气 | 筑基境 | — | 解锁本命飞剑养成 |
| 练气 | 金丹境 | — | 解锁金丹神通 |
| 练气 | 元婴境 | — | 解锁元神出窍 |
| 练气 | 化神境 | — | 解锁领域 |
| 练气 | 飞升境 | — | 解锁天人感应 |
| 终极 | 十五境 | — | 通关成就 |

### 突破机制
- 每个小境界：灵气 + 悟性达标即可突破（成功率100%）
- 大境界突破：需要灵气达标 + 根基达标 + 渡劫（有失败概率）
- 渡劫失败：受伤掉属性，但不会死亡，可重新积累再突破
- 武道→练气的关键跨越：需要"开窍"事件触发

## 三、核心系统（★ = 主要系统）

### ★1. 修炼系统（核心成长）

**修炼方式：**
- **吐纳打坐**：稳定获得灵气 + 锤炼根基，受功法加成
- **读书悟道**：消耗"悟性"值，获得修炼速度加成（对应书中"读书人"设定）
- **剑术修炼**：消耗体力，提升剑术熟练度

**属性面板：**
- 灵气（当前 / 上限）— 突破门槛
- 根基（当前 / 上限）— 境界稳定性，根基不足渡劫必败
- 悟性 — 影响读书悟道效率、剑术学习速度
- 剑意 — 影响战斗伤害，特殊剑招需要剑意门槛
- 体魄 — 影响气血上限、防御
- 气血 — 战斗中消耗，归零则重伤
- 神识 — 影响秘境探索、发现隐藏剧情

### ★2. 剑术/功法系统

**功法体系：**
- 按品质分：凡品 → 灵品 → 仙品 → 神品
- 每部功法有等级（1-10重），修炼到满重可解锁被动效果
- 功法来源：书院藏书、宗门传承、秘境发现、奇遇

**剑术体系：**
- 基础剑招：人人可学（砍、刺、撩、格挡等）
- 上乘剑术：需要对应功法支撑（如《养剑经》《止境剑诀》等书中剑法）
- 本命飞剑：伴随玩家成长的佩剑，可升级、开锋、养剑意
- 剑术等级：入门 → 熟练 → 精通 → 大成 → 圆满

### ★3. 战斗系统（自动战斗）

**战斗模式：**
- 自动进行，无需操作
- 双方数据对比 + 随机波动 = 胜负判定
- 战斗过程用文字描述（一两句话）

**胜负因子：**
- 境界差距（最大权重）
- 剑术等级 vs 对手
- 法宝/符箓加成
- 体魄（气血量）
- 功法克制（如克制关系 +20%伤害）
- 运气波动（±10%）

**战斗结果：**
- 胜：获得灵气 + 修为值 + 概率掉落物品
- 负：损失部分灵气 + 一定概率掉落物品（不会死亡，而是"重伤休养"）

### 4. 炼剑/养剑系统

**本命飞剑：**
- 初始：一把普通铁剑（可命名）
- 升级途径：吸收灵矿 → 提升品阶（凡兵→灵兵→仙兵→神兵）
- 开锋：需要特定机缘事件触发
- 剑意养成：使用同一把剑战斗越久，剑意越强（类似书中"人与剑心意相通"）

**剑匣系统：**
- 可放置最多3把备用飞剑
- 战斗中可自动切换（根据对手属性）

### 5. 地图探索系统

**地图结构：**
- 浩然天下 → 各州（宝瓶洲、中土神洲、北俱芦洲等）
- 随境界解锁新区域
- 每个区域有：可探索地点 + 随机事件 + 势力任务

**探索方式：**
- 消耗"行动力"（每天回复一定量）
- 选择目的地 → 系统生成事件（安全修炼 / 偶遇奇遇 / 遭遇战斗）
- 秘境：特殊副本，需要神识门槛进入

### 6. 宗门/势力系统

**可选势力（书中名门正派）：**
- 泥瓶巷（初始，陈平安阵营）→ 剑气长城
- 书简湖 / 学塾
- 道家 / 佛家 / 儒家（三教可选）
- 各洲大宗门

**势力内容：**
- 宗门任务（悬赏、巡逻、采集）
- 宗门贡献 → 兑换功法/资源
- 宗门地位（外门→内门→真传→长老）

### 7. 背包/装备系统

**物品分类：**
- 剑器（主武器）
- 法袍（防御）
- 符箓（一次性道具，战斗中自动使用）
- 丹药（恢复气血、临时增幅）
- 灵材（升级剑器用）
- 书籍（功法、剑谱）

### 8. 剧情事件系统

**触发方式：**
- 境界突破时触发关键剧情（对应原著重要节点）
- 探索时随机触发书中经典桥段（如"泥瓶巷初遇宁姚"）→ 获得特殊奖励
- 势力任务线推进 → 解锁后续剧情

**设计原则：**
- 不剧透未读章节，只使用已公开内容
- 事件是"重温/体验"而非"剧透"
- 经典选择让玩家做决定（如帮/不帮某人），影响后续事件走向

### 9. 随机奇遇系统

**奇遇类型：**
- 灵雨/药园/秘境 → 资源收获
- 古碑/残卷 → 功法或剑术
- 心魔 → 考验，过则属性提升，不过则受伤
- 灵兽结缘 → 获得灵兽助战（可做，轻量）
- 论道 → 与其他修士（NPC）交流，获悟性

## 四、UI设计

**页面结构（沿用问道长生竖屏框架）：**

顶部：角色概览（境界、灵气、气血）
主页面：当前行动 + 状态摘要 + 修行日志
底部导航（5个标签）：

1. **主页**（修炼、打坐、悟道）
2. **行旅**（探索地图、秘境）
3. **剑阁**（剑术、功法、飞剑养成）
4. **洞府**（背包、装备、势力）
5. **道册**（境界、属性详情、图鉴）

## 五、数值设计（初期策划）

### 境界灵气需求（示例）

| 境界 | 需要灵气 | 需要根基 | 其他条件 |
|------|---------|---------|---------|
| 铜皮1层 | 100 | — | 锻体功法1重 |
| 铜皮3层→锻体1层 | 1000 | 300 | 锻体功法3重 |
| 锻体3层→归真 | 5000 | 1500 | 内功心法 |
| 归真→练气1层 | 20000 | 8000 | **开窍事件** |
| 练气1→9层 | 逐层递增 | — | 每层需剑术等级 |
| 练气9→筑基 | 20万 | 5万 | 渡劫成功 |
| 金丹 | 100万 | 30万 | 渡劫 |
| 元婴 | 500万 | 150万 | 渡劫+本命飞剑灵兵以上 |
| 化神 | 2000万 | 800万 | 渡劫+功法至少仙品 |
| 飞升 | 1亿 | 3000万 | 渡劫+剑意大成 |
| 十五境 | 5亿 | 1亿 | 全图鉴收集+全剧情通关 |

### 修炼效率

| 方式 | 灵气/次 | 消耗 | 备注 |
|------|---------|------|------|
| 吐纳（基础） | 10~20 | 无 | 稳定但慢 |
| 吐纳（有功法） | 20~60 | 体力 | 功法加成 |
| 读书悟道 | 0 | 悟性 | 提升修炼速度% |
| 战斗胜利 | 50~200 | 气血 | 快但危险 |
| 丹药 | 500~5000 | 灵材 | 消耗品 |

## 六、技术方案

- **框架**：纯 HTML/CSS/JS（同问道长生，单页应用）
- **存储**：localStorage（自动存档）
- **离线**：已配置 Service Worker + Manifest（直接复用 PWA 封装）
- **UI**：竖屏底部导航 + 顶部状态栏（同问道长生框架）
- **开发路径**：先完成核心循环（修炼→突破→战斗），再逐步扩展次要系统

## 七、版本规划

### v1.0 — 核心循环
- 境界系统（凡人→练气→筑基）
- 修炼系统（吐纳、悟道）
- 基础剑术（3~5种剑招）
- 自动战斗系统
- 基础背包
- 一个可探索区域（宝瓶洲·泥瓶巷）

### v1.1 — 内容扩展
- 炼剑/飞剑养成
- 更多区域解锁
- 势力系统
- 剧情事件

### v1.2 — 完整版
- 全部境界（至飞升）
- 所有功法/剑术
- 秘境系统
- 随机奇遇
- 势力完整任务线
- 图鉴系统
