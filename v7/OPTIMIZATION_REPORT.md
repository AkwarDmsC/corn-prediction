### 2. 震荡熔断优化（decider.py）

旧规则：`total_active <= 1` 直接强制震荡
新规则：`total_active == 1` 且 `|score| < 0.10` 才熔断

效果：**32.8%** 的样本保留单信号强方向判断（旧规则 0%）。

### 3. 新增做多因子（signals.py）

- BB_LOWER_TOUCH：布林带下轨 + 均线不显著多头 → 看涨
- CONSECUTIVE_DOWN：连续下跌 >= 3 天 → 均值回归看涨
- GAP_DOWN：跳空低开 > 0.3% → 缺口修复看涨

**重要发现**：2023-2026 做多因子本身准确率仅 44-45%（低于自然上涨率 48%），因此大幅降权到 0.3。做空偏重是**市场状态导致的**（近 3 年玉米震荡偏空），非信号体系问题。

## 信号质量评估总结

**真正的瓶颈不是信号权重，而是信号源太少。** v7 只有 3 个有效信号（MA/RSI/BB），在扁平震荡市几乎全部为 0。2023-2026 玉米的日波动率太低（ATR20 仅 0.8%），导致超买超卖、布林极值等触发率极低。
