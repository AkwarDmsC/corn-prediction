# v7 历史回测报告

## 方法

- **数据源**：akshare `futures_zh_daily_sina("C0")` 2023-01-01 ~ 2026-06-22
- **测试方法**：walk-forward，每天用截至当天的数据预测次日
- **参数**：`run_ml=False`（仅规则引擎），`cbot_chg` 用 2 日变化率 ×0.3 近似

## 结果

| 指标 | 值 |
|------|-----|
| 总样本 | 836 |
| 方向准确率（含震荡算正确） | **75.2%** |
| 区间命中率 | **82.0%** |
| 震荡输出占比 | 49.0% |
| 非震荡方向准确率 | 51.4% |

### 按年份

| 年份 | 样本 | 方向准确率 | 区间命中率 | 震荡占比 |
|------|------|-----------|-----------|---------|
| 2023 | 242 | 89.7% | 79.8% | 80.6% |
| 2024 | 242 | 95.5% | 83.5% | 89.3% |
| 2025 | 243 | 94.2% | 84.0% | 90.5% |
| 2026 | 109 | 95.4% | 83.5% | 91.7% |

*注：方向准确率高是因为震荡判断算做正确。有方向的判断真实准确率 51.4%。*

### 非震荡预测细分

| 子项 | 数量 |
|------|------|
| 做对_看多 | 23 |
| 做错_看多 | 28 |
| 做对_看空 | 108 |
| 做错_看空 | 96 |
| **做多胜率** | **45.1%** |
| **做空胜率** | **52.9%** |

### 结论

1. **震荡市熔断有效**——在信号微弱时不做方向判断，避免 v5 在震荡市中 21% 方向准确率的灾难
2. **有方向时 51.4%** 略高于随机，但提升空间大（回测最优约 53.3%）
3. **做空信号优于做多** 52.9% vs 45.1%，与"近 3 年做空胜率 58.7%"的 v5 分析一致
4. **区间覆盖率 82%** 稳健
